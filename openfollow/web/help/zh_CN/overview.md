# 服务器网络

局域网上发现的所有 OpenFollow 站点的实时视图。演出前确认所有站点在线且可访问。

每行代表一个站点——本服务器或通过组播发现的对等站点。

- **状态指示器** – 实心圆 (●) = 在线且响应；空心圆 (○) = 之前见过但当前不可达。
- **站点名称** – 在 通用 → 站点设置 中设置的名称。本服务器的行标注为 *(本服务器)*。
- **地址** – 该站点服务的 IP 地址和 Web 端口（如 `192.168.1.42:80`）。在浏览器中打开可访问该站点的 Web 界面。

> 如果没有显示对等站点，请检查其他 OpenFollow 实例是否正在运行且在同一网段。组播对等发现不能跨越路由器跳数或 VLAN 边界。

面板每 5 秒自动刷新。使用**刷新**可立即更新——例如刚启动第二台站点后。
