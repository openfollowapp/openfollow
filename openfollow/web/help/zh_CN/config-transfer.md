# 配置迁移

将站点的完整设置备份和恢复为单个 `.openfollowsettings` 文件——跨站点复制配置、在修改前归档、或从快照恢复。

**导出配置** – 将当前配置下载为 `.openfollowsettings` 文件，包含所有保存的设置（相机、网格、标记、触发区域、OSC、MIDI、输入、显示）。

**配置文件 (.openfollowsettings)** – 选择之前从任意 OpenFollow 站点导出的 `.openfollowsettings` 文件；应用前会进行验证。

**导入配置** – 应用所选文件。站点的网络 IP 被保留；其他所有设置被替换。如果任何导入的设置需要重启，会先显示确认对话框；否则页面自动重载。

重启对话框提供三个选项：

- **立即重启** – 应用所有内容并重启；站点重新上线后页面自动重载。
- **不重启应用** – 应用可立即生效的部分；其余等待下次重启。
- **取消** – 放弃导入；不改变任何内容。
