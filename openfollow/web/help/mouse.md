# Mouse Input

> Mouse Input is marked **Experimental**. It is off by default and not recommended as a primary input method during live shows.

Click-and-drag control for positioning markers directly on the live video. When enabled, a click on the Operator Screen locks onto that position and dragging moves the active marker in real time. Useful for static positioning during programming when a gamepad or keyboard isn't to hand.

A **left-click both arms control and snaps the active marker to the cursor in that same click** – there is no separate "arm, then move" step, so any click on the Operator Screen repositions the marker immediately. **Right-click** disarms control (movement stops until the next left-click). The **scroll wheel** adjusts the marker's height – scroll **up to raise** it, down to lower it. Because a stray click moves the marker, leave Mouse Input off unless you're actively placing markers.

**Enabled** – turn mouse input on or off. When disabled, clicks on the live overlay have no effect on marker positions and the cursor is hidden from the display immediately. Turning it off mid-session also disarms control, so re-enabling it won't move the marker until you click again.

**Save** – store the current setting. The change applies immediately but is lost on restart unless you save.
