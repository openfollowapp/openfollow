# 网络设置

站点的 IPv4 配置——与设备端**设置 → 网络**界面相同的设置。

**接口** – 要配置的适配器（通常为 `eth0`）。切换时重新加载该适配器的表单。

**方式**：

- `DHCP（自动）` – 路由器分配所有参数。
- `DHCP 手动地址` – DHCP 提供子网和路由器；你固定 IP。
- `静态` – 自行输入地址、子网掩码和路由器。

**地址字段**（始终显示；在 `静态` 或 `DHCP 手动地址` 模式下可编辑）：

- **IP 地址** – 本机 IPv4 地址；用于 PSN 输出、对等发现和 Web 服务器。
- **子网掩码** – `静态` 模式必需；`DHCP 手动地址` 模式下从租约获取。
- **路由器**（可选）– 默认网关；无互联网连接的局域网可留空。必须在子网范围内，否则拒绝。

**DNS（服务器 1–3）** – 按优先级排序的解析器地址。仅在需要访问外部主机名时需要（如软件更新）；离线局域网可留空。

**租约剩余时间** – 当前 DHCP 租约倒计时（只读，每 5 秒刷新）。

> 应用可能导致当前 Web 会话断开。静态/手动地址会自动在新地址重新加载 UI；DHCP 模式下如果会话断开请手动重连。

**模式：** 表单以**查看模式**打开——字段锁定防止误改。使用**切换到编辑视图**解锁；**编辑模式**显示 应用 / 续租 / 取消。如果站点的网络后端为只读，表单显示**只读**标签——请从设备端**设置 → 网络**菜单配置，或访问 openfollow.app 查看如何启用 Web 编辑。

**按钮：**

- **切换到编辑视图** – 解锁字段（进入编辑模式）。后端只读时不可用。
- **应用** – 验证并提交表单。无效输入被拒绝且不写入任何内容。
- **续租 DHCP** – 请求新租约（仅 DHCP 模式）。
- **取消** – 丢弃未保存的编辑并返回查看模式。
