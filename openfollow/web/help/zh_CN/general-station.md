# 站点设置

本机的标识、显示偏好和 Web 访问设置。此处的更改立即生效——无需重启。

**单位系统** – 选择**公制（m, m/s）**或**英制（ft / in, ft/s）**。这控制 Web UI 和操作员屏幕在相机、网格、标记、运动、触发区域和设置向导中的显示和解析方式。存储的配置和所有有线协议——OSC、PSN、RTTrPM、OTP——始终使用公制，与此设置无关。更改后立即生效，无需保存。

**显示实验性功能** – 默认关闭。启用后显示仍在开发中的早期/粗糙功能：**人形检测**（独立标签页）、**鼠标输入**（输入标签页）和 **RTTrPM 输出**（输出标签页），各功能标有*实验性*徽章。切换后立即生效——无需重载。将其**关闭**也会禁用鼠标输入和人形检测，防止隐藏的功能继续运行；重新打**开**不会自动重新启用它们——需在各自设置区手动重新开启。

**站点名称** – 该设备的人类可读名称，最多 64 个字符。显示在 Web UI 标题栏、总览标签页的服务器网络列表中，并作为 PSN 系统名称广播给网络上的其他演出控制工具。默认为 `OpenFollow`；首次启动时自动追加一个易记的双词后缀，使多台站点保持可区分。清空字段将恢复默认值。

**PIN（留空禁用）** – 数字 PIN（1–32 位），保护 Web UI 中的所有配置路由。未设置时，接口对网络上的任何人开放。设置后，浏览器必须提供 PIN 才能访问任何非资源页面，OpenFollow 站点间的对等配置交换也使用它进行认证（PIN 本身不在线路上传输）。

> 在连接到共享制作网络的任何站点上设置 PIN。仅在隔离的测试台或点对点网络上可以不设置。

**保存** – 将站点名称和 PIN 写入磁盘。两个字段保存后立即生效，无需重启应用。
